# 📝 Interactive Task Manager

## 📌 Project Description

The Interactive Task Manager is a feature-rich web application built using vanilla JavaScript.  
It allows users to create, update, delete, and manage tasks efficiently with data persistence using localStorage.

The application demonstrates DOM manipulation, event handling, modular JavaScript architecture, and responsive UI design.

---

## 🚀 Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as complete/incomplete
- Filter tasks (All, Active, Completed)
- Task statistics (Total, Active, Completed)
- Clear completed tasks
- Dark/Light mode toggle
- Task priority levels (Low, Medium, High)
- Due date support
- Data persistence using localStorage
- Modular JavaScript file structure

---

## 📊 Sample Output

A functional task manager with:

- **15 Total Tasks**
- **5 Completed Tasks**
- **10 Active Tasks**

Filtering options:
- **All** → Displays all 15 tasks
- **Active** → Displays 10 active tasks
- **Completed** → Displays 5 completed tasks
