# 📚 Console-Based Library Management System

A Java console application that manages books and members using Object-Oriented Programming and file handling for persistent storage.

This project demonstrates backend fundamentals such as class design, collections, file I/O, and date handling without using a database.

---

## 🚀 Features

- Add new books
- View all books
- Search books
- Register members
- Borrow books
- Return books
- View library statistics
- Persistent data storage using text files
- Due date management using `LocalDate`

---

## 🛠️ Technologies Used

- Java 17
- Object-Oriented Programming (OOP)
- ArrayList
- BufferedReader & BufferedWriter
- java.time.LocalDate

---

---

## ⚠️ Important Notes

- Ensure `books.txt` and `members.txt` are inside the `data` folder.
- Recompile the project after making any changes.
- The system automatically updates data files when books are borrowed or returned.
- If data files are missing, the system starts with empty records.
