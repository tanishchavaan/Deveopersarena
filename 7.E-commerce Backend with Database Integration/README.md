# E-commerce Backend with Database Integration

A comprehensive e-commerce backend system with complete database integration.

## Features
- PostgreSQL database with proper schema design
- Spring Data JPA repositories with custom queries
- Transaction management for order processing
- Database migrations with Flyway
- Auditing with Spring Data JPA auditing
- Query optimization with indexes and fetching strategies
- Connection pooling with HikariCP

## Running the application
1. Start infrastructure: `docker compose up -d`
2. Run application: `./mvnw spring-boot:run`
