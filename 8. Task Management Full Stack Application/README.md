# 🚀 Task Manager Full Stack Application

A complete **Full Stack Task Management Application** built with **React (TypeScript)** and **Spring Boot**, featuring authentication, real-time updates, and a modern responsive UI.

---

## 🌟 Features

### 🔐 Authentication
- JWT-based authentication (Access + Refresh Tokens)
- Secure login & registration
- Protected routes

### 📋 Task Management
- Create, update, delete tasks
- Drag-and-drop task status (Kanban board)
- Task filtering, sorting, and search
- Task priorities & due dates

### 💬 Task Details & Comments
- Detailed task view
- Add and view comments per task

### ⚡ Real-Time Updates
- WebSocket-based live updates
- Instant task status synchronization

### ⏰ Reminders
- Scheduled notifications for due tasks

### 🎨 UI/UX
- Responsive design (mobile + desktop)
- Dark/Light mode toggle
- Keyboard shortcuts for productivity

### 🧪 Testing
- Backend unit & integration tests (JUnit, Mockito)
- Frontend component testing
- End-to-End testing (Cypress/Playwright)

---

## 🛠️ Tech Stack

### Frontend
- React + TypeScript
- Vite
- Tailwind CSS / Material UI
- Axios
- Zustand / Context API

### Backend
- Spring Boot
- Spring Security + JWT
- Spring Data JPA
- WebSocket
- PostgreSQL

### DevOps
- Docker & Docker Compose
- CI/CD (GitHub Actions)
- Deployment (Vercel / Railway / Render)
